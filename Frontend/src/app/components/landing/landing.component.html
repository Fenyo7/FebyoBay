Welcome to GreenBay!

GreenBay is a simple eBay clone where users can sign in to sell and buy items. Explore a wide range of items available for sale, or list your own items to reach potential buyers.

Features

  • Register to the application (not implemented yet)
  • Sign in to the application (not implemented yet)
  • Create a sellable item
  • List existing sellable items
  • View a specific sellable item's details
  • Buy a sellable item (not implemented yet)

Technical Specifications

The application's backend is a JSON-based Web API, and the frontend is developed using Angular. Authentication is secured using JWT tokens, and data is stored in a relational database.
